Tax Academy Singapore

eLearning Video Streaming Platform

This repository hosts the full-stack source code for the Tax Academy Singapore eLearning video streaming platform, including frontend, backend, database schema, and infrastructure configuration.

To run the application, please refer to the project README and setup documentation in the docs/ folder.

Key capabilities include subscription plans, pay-per-view access, live webinars, and free educational video content, with an admin dashboard for content and user management.
